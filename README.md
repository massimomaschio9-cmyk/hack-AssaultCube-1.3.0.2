# AssaultBot 1.3.0.2

Un trainer educativo scritto in Python per AssaultCube v1.3.0.2. Questo progetto esplora la manipolazione della memoria e l'overlay grafico per scopi di studio.

## Caratteristiche
* **Aimbot**: Aggancio automatico del bersaglio.
* **ESP Boxes & Lines**: Visualizzazione dei nemici attraverso i muri.
* **God Mode**: Salute infinita.
* **Infinite Ammo**: Munizioni illimitate.
* **Mass Kill & Teleport**: Funzioni avanzate di manipolazione delle entità.

## Requisiti
* Python 3.x
* Librerie: `pymem`, `keyboard`, `pywin32`

## Installazione
1. Installa le dipendenze: `pip install pymem keyboard pywin32`
2. Avvia AssaultCube (Modalità Finestra).
3. Esegui il terminale come Amministratore.
4. Avvia il bot: `python Main.py`

## Disclaimer
Questo progetto è creato solo a scopo educativo. L'uso di cheat nei server pubblici può portare al ban.
